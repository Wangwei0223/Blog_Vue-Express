CSS居中方法总结

左右等宽 + 中间自适应 第一种:绝对定位

三列等宽左右宽从定长改成百分比即可 需要给中间加padding不然会遮挡左右

中间若是在上方, 左右position:absolute后要 top:0把给中间腾出的位置覆盖上

左边 
#left_1, #right_1{
    background-color: aqua;
    position: absolute;
    width: 200px;
    height: 100%;
    top: 0;
}
右边 
#right_1{
    right: 0;
}
中间 
#main_1{
    text-align: center;
    background-color: blueviolet;
    height: 100%;
    padding-left: 200px;
    padding-right: 200px;
}

左右等宽 + 中间自适应 第二种:float+负边距 (圣杯) 不知道为什么好多面试考,感觉很麻烦的布局

第一点: float和position:absolute的初始地点:父级左上,但是重点是 如果父级有一个div占了一行,并且这行还在float和position上方,会给它腾地方

第二点: float和position:absolute之后宽度变为内容的实际宽度,不再是一整行

第三点: 这么玩中间必须在上方,不然重新调

中间 
#main_2 {
    text-align: center;
    float: left;
    width: 100%;
    height: 100%;
    background-color: blueviolet;
}
左边 
#left_2,
#right_2 {
    position: relative;
    background-color: aqua;
    width: 200px;
    height: 100%;
    float: left;
}

#left_2{
    margin-left: -100%;
    left: -200px;
}
右边 
#right_2 {
    margin-left: -200px;
    right: -200px;
}

左右等宽 + 中间自适应 第三种:float + 负边距(双飞翼)

float:left 不会重叠, position:absolute 会重叠

中间 
#main_3{
    text-align: center;
    float: left;
    width: 100%;
    background-color: blueviolet;
    height: 100%;
}

#main_3 #content{
    margin-left: 200px;
    margin-right: 200px; /*padding效果也一样*/
}
左边 
#left_3, #right_3{
    float: left;
    width: 200px;
    height: 100%;
    background-color: aqua
}    

#left_3{
    margin-left: -100%;
}
右边 
#right_3{
    margin-left: -200px;
}

左右等宽 + 中间自适应 第四种:只用 float

缺点就是最小的时候会错位 规避方法就是中间也浮动,负边距再调(回到双飞翼)

注意以上全部方法父级position都不能为relative, 不然不会被覆盖

相对定位是不能层叠的!占着原来的位置!

左边 
#left_4, #right_4{
    float: left;
    width: 200px;
    height: 100%;
    background-color: aqua;
}
右边 
#right_4 {
    float: right;
}
中间 
#main_4 {
    background-color: blueviolet;
    height: 100%;
    padding: 0 200px 0 200px;
}

左右等宽 + 中间自适应 第五种:table table-cell

秒啊! 啥都不用调 什么position:absolute float:left 负边距 全不用管!天生自适应还没覆盖问题!

唯一要注意的是外层加table之后 width会变为实际占的大小,需要加width:100%调回来 (和float和absolute一样, 注意relative不会消失)

左边 
#left_5, #right_5{
    display: table-cell;
    background-color: aqua;
    height: 100%;
    width: 200px;
}
中间 
#container_5{
width: 100%;
display: table;
}

#main_5{
    display: table-cell;
    text-align: center;
    background-color: blueviolet;
    height: 100%;
}
右边 
#left_5, #right_5{
    display: table-cell;
    background-color:aqua;
    height: 100%;
    width: 200px;
}
