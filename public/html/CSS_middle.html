CSS居中方法总结

以下方法父级均没有垂直居中,因为父级的父级没有设置长宽,会塌陷, 只用auto实现了水平居中

第一种 + 第二种 父级: margin:auto 子级: 负边距

     
                                #parent_1 {
                                    position: relative;
                                    background-color: aqua;
                                    width: 200px;
                                    height: 200px;
                                    margin: auto;
                                }
                                                       
                                #child_1{
                                    position: absolute;
                                    background-color: blueviolet;
                                    width: 100px;
                                    height: 100px;
                                    left: 50%;
                                    top: 50%;
                                    margin-left: -50px;
                                    margin-top: -50px;
                                }

第三种 父级:同上 子集:transform: translate(-50%, -50%)

                           
                                #child_2 {
                                    position: absolute;
                                    background-color: blueviolet;
                                    width: 100px;
                                    height: 100px;
                                    left: 50%;
                                    top: 50%;
                                    transform: translate(-50%, -50%);
                                    font-size: 15px;
                                }

第四种 父级:table 子集:table-cell

     
                            #child_3 {
                                display: table-cell;
                                background-color: blueviolet;
                                text-align: center;
                                vertical-align: middle;
                            }
                        
                            #child_4 {
                                display: table-cell;
                                background-color: pink;
                                text-align: center;
                                vertical-align: middle;
                            }
                            display: table-cell 就是把元素以td一样的形式呈现
                            而td是行内,或者说特殊的行内
                            行内是什么样？display:inline; 此时，元素p hello p的宽高只受内部文字影响
                            要百分比响应父元素的宽高，只能block、inline-block
                            
                            所以垂直居中个文字还行,想上面两个同心 div 做不到
